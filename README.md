# Coffee-Order-Excel
Interactive Excel dashboard analyzing coffee sales trends across products, countries, and customers (2019–2022).
A multi-sheet Microsoft Excel project analyzing 1,000+ coffee orders across four product types — Arabica, Excelsa, Liberica, and Robusta. Built dynamic pivot tables and charts to visualize monthly and yearly revenue trends, country-wise performance (US, Ireland, UK), and top customer rankings. Designed an interactive dashboard summarizing key business insights to support data-driven decision making.
